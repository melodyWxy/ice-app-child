# ice-stark-child

## 作为子应用的连调方式
参照文档： https://github.com/melodyWxy/ice-app-container/blob/master/README.md

## 使用

- 启动调试服务: `npm start`
- 构建 dist: `npm run build`

## 目录结构

- 应用配置: `src/app.js`
- 路由配置: `src/routes.js`
- 布局文件: `src/layouts`
- 通用组件: `src/components`
- 页面文件: `src/pages`
